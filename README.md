# Defi Pair Monitor

DeFi Pair Monitor is a tool designed to detect and monitor the creation of new token pairs on various blockchains, such as Ethereum, Arbitrum, and others. This tool helps users keep track of new opportunities in the DeFi ecosystem, such as providing liquidity, arbitrage, or finding new projects.

# Features
- Supports monitoring multiple blockchains
- Easily configurable through a JSON configuration file
- Extensible to support more blockchain networks and decentralized exchange platforms
- Written in Go for high performance and concurrency
